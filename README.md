## External dependencies
* Node.js
* npm
* MongoDB
* Redis
* GraphicsMagick (for trimming, compress, etc etc)

## How to build
#### step by step
1. `git clone git://github.com/misskey-delta/Misskey-Web.git`
2. `cd Misskey-Web`
3. `npm install`
4. `npm run bower`
5. `npm run build`

#### one shot
`npm run deploy`

## How to start Misskey Web server
`npm start`

## How to display options
`npm start -- -h`
